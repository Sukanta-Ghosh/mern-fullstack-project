# MERN-Fullstack-Project

### Create Project:

`npm init -y` -> Create package.json
`npm i express`
`sudo npm i nodemon -g`

### Run Project:

`node fileName.js`
or
`nodemon fileName.js`
or
`npm run start` if configured on package.json
